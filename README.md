# 贪吃蛇游戏 🐍

基于 Python + Turtle 库开发的多模式贪吃蛇游戏，支持无尽模式和关卡模式。

## 游戏特性

- **双模式支持**: 无尽模式（30x30大地图）+ 4个特色关卡
- **彩虹变色**: 无尽模式300分后触发彩虹变色效果
- **速度递增**: 随分数增长自动加速，增加挑战性
- **进度保存**: 自动保存已解锁关卡进度
- **精美界面**: 海龟图形绘制，网格背景，边框提示

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| Enter / 空格 | 开始游戏 |
| P | 暂停/继续游戏 |
| R | 重新开始当前游戏 |
| Esc | 返回主菜单 |

## 游戏模式

### 无尽模式
- 地图尺寸：30x30
- 目标：无限挑战，追求高分
- 吃到食物得分：10分/个
- 速度随分数增长：每达到一定阈值加速一次
- 变色机制：
  - 0分：白色
  - 20分：绿色
  - 50分：蓝色
  - 90分：紫色
  - 140分：金色
  - 200分：红色
  - 300分：彩虹模式（蛇身七彩色）

### 关卡模式

| 关卡 | 名称 | 地图 | 速度 | 目标分数 | 特性 |
|------|------|------|------|----------|------|
| 1 | 🟢 新手训练营 | 20x20 | 0.6x | 100分 | 熟悉操作 |
| 2 | 🟡 迷宫探险 | 24x24 | 0.75x | 150分 | 绕过障碍 |
| 3 | 🔵 限时挑战 | 24x24 | 0.9x | 120分 | 45秒倒计时 |
| 4 | 🔴 终极考验 | 26x26 | 1.0x | 200分 | 随机障碍物 |

**关卡解锁**：通过当前关卡后自动解锁下一关

## 游戏规则

1. **移动**：蛇自动向当前方向移动，玩家通过方向键改变方向
2. **吃食物**：蛇头碰到食物时，身体增长一节，得10分
3. **死亡条件**：
   - 头部撞墙
   - 头部碰到自身身体
   - 碰到障碍物（关卡模式）
